# Hello API

Boilerplate project to have a GO REST API Template along with 
fully functioning CI/CD pipeline.

## Dependencies

- Go version 1.20.5


## Setup

### Install Go

`sudo make setup`

### Upgrade Go

`sudo make install-go`
